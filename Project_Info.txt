FW Launcher

-Задачи программы:

--Запуск игры Forsaken World
--Автоматический ввод данных аккаунта пользователя
--Локальное хранение данных аккаунта пользователя
--Запуск игры в несколько окон

-Функционал программы

--Главное окно

---Список аккаунтов

----Отображение добавленных аккаунтов
----Данные аккаунта хранятся локально в реестре компьютера пользователя и нигде в списках не отображаются (кроме самого названия аккаунта)
----Возможность выбора от 1 до 3-х аккаунтов для последовательного их запуска с воддом хранящихся данных.

--Кнопка Менеджер аккаунтов

---Менеджер аккаунтов

----Позволяет ввести данные аккаунта (имя, логин для входа в игру, пароль для входа в игру)
----Список аккаунтов отображает добавленные аккаунты и синхронизирован со списком аккаунтов в главном окне.
----После выхода из FW Launcher список аккаунтов сохраняется и не пропадает.
----Имеется функция изменения данных аккаунта, для исправления ошибок при вводе данных пользователем. Изменения вступают в силу сразу после нажатия на соответствующую кнопку.
----Функция удаления аккаунта, которая позволяет удалить аккаунт как сосвсех списков, так и из реестра.
----Функция удаления всех аккаунтов, активирующаяся после добавления 2-го и более аккаунта в список. Данная функция введена для удобства удаления аккаунтов по инициативе пользователя.
----Кнопка возврата в главное окно.
----Функция отмены действий при нажатии на клавишу Esc
----Функция скрытия введенного пароля для конфидециальности ввода данных пользователем.
----Кнопка показа пароля, для контроля введенных данных.
----Интерактивная справка

-----Справка представляет собой интерактивное обучающее пособие для тех, кто хочет подробно узнать информацию о работе с FW Launcher (далее Программа).
-----Несколько вкладок демонстрируют каждое определенное действие при работе с Программой.
-----Вкладки включают в себя: Добавление, изменение, удаление аккаунтов.
-----Для удобства изначально в списке аккаунтов находится справочный аккаунт, который может подвергнутся изменению/удалению со стороны пользователя, с последующим восстановлением своих данных для практики работы с Программой.
-----Имеется возможность переключаться между вкладками ка с помощью мыши, кликая на необходимые вкладки, так и с помощью кнопок расположенных в нижней части программы.
-----Кнопка возврата в менеджер аккаунтов

--Кнопка настроек

---Настройки программы

----Выставление и хранение пауз, которые используются при вводе данных пользователя в лаунчер игры.
----Кнопка сохранения введенных пауз.
----Кнопка выставления пауз по умолчанию, рекомендуемых даже для слабых компьютеров.
----Возможность ввода пути до исполняемого файла игры, с контролем недопустимости неправильных действий.
----Возможность добавления пути в реестр, для хранение данной информации после выхода из Программы.
----Возможность изменения пути игры при определенных ситуациях (таких как переустановка игры в другое место).
----Возможность удаления пути игры.
----Кнопка удалить ВСЕ данные подготавливает Программу к полному удалению. Это сделано с расчетом, чтобы очистить реестр от всех следов работы программы и легкого удаления Программы пользователем. После нажатия на кнопку, Программа закроется, все данные с реестра будут удалены, а Программу останется только удалить в корзину.
----Возможность сохранения выбранных аккаунтов данные которых будут введены при запуске игры. При включенной функции выбранные аккаунты будут автоматически выбираться при запуске Программы.
----Кнопка возврата в Главное окно.
----Кнопка Справки

-----Интерактивная справка способствует практическому изучению функционала окна настроек без изменения самих настроек Программы.
-----Переключение между вкладка осуществляется самостоятельным кликом пользователя по определенной вкладке, нажатием на кнопки расположенные внизу Программы, а так же функцией автоматического переключения вкладок, после достижения определенных целей действующей вкладки.
-----Вкладка Добавить путь полностью симулирует добавление пути до испольнительно файла игры.
-----Вкладка изменение так же симулирует изменение пути игры. Если пользоваль не имеет возможности изменить путь на аналогичный, предоставлена кнопка "Схитрить", которая создает нужную диреторию с необходимым фалом для изменения, удаляемый кнопкой Удалить путь.
-----Вкладка Удалить путь производит симуляцию удаления пути, а так же удаляет директорию с файлом, созданную кнопкой Схитрить.

--Start FW осуществляет запуск игры в новом потоке (без данной функции, программа не действола на компьютере со слабой конфигурацией), с последовательным вводом данных пользователя с учетом пауз.
--Кнопка выхода из Программы.
